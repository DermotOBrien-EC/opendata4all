# Code of Conduct

opendata4all follows the Contributor Covenant Code of Conduct, version 2.1:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

Project spaces must stay usable for contributors, donors, researchers, and
reviewers who are handling sensitive material. Harassment, coercive data
collection, attempts to expose private information, and pressure to waive
privacy rights are not acceptable.

Report conduct issues to the maintainers listed in the project governance
documents. Security or privacy incidents should follow [SECURITY.md](SECURITY.md).
